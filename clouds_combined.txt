

https://cds.climate.copernicus.eu/cdsapp#!/dataset/reanalysis-era5-single-levels

```
import cdsapi

c = cdsapi.Client()

c.retrieve(
    'reanalysis-era5-single-levels',
    {
        'product_type': 'reanalysis',
        'variable': [
            'high_cloud_cover', 'low_cloud_cover', 'medium_cloud_cover',
        ],
        'year': '2021',
        'month': '01',
        'day': '02',
        'time': '07:00',
        'format': 'grib',
    },
    'download.grib')
```
